# Nextest

A partially API-compatble sever alternative to Nextcloud, without all the fuzz.

**Not production ready.**

## Why?

Self-hosted file storage is complicated. Nextcloud is what should make it easy,
at least from what I've heard. Sadly, it requires a database, uses a "lot" of
memory, and is somewhat slow (from my experience). I wanted something that was
simpler to set up and can run on a Raspberry Pi. I also wanted something that is
compatible with the Nextcloud API (OCS), so that I can use existing Nextcloud
apps for sync.
